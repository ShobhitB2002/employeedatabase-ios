EmployeeDatabase-iOS

An iOS app that manages employee data using Core Data.  
Built in MVC.

Features
- Add, edit, and delete customer data
- Core Data integration
- Clean architecture : MVC

Technologies
- Swift
- UIKit
- Core Data
- MVC

Getting Started
1. Clone this repo
2. Open `EmployeeDatabase.xcodeproj`
3. Run the app
